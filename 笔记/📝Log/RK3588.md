开机自运行

首先打开编辑界面
```shell
gnome-session-properties
```

然后点【Add】，在弹出框中，【Name】随便起一个，【Command】的格式为：
```shell
gnome-terminal -x /path/to/my_script.sh
```
加上`gnome-terminal`是为了开机在当前窗口自动打开一个终端
