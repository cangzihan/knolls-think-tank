## Nginx 是什么？
Nginx 是一个高性能的 Web 服务器和反向代理服务器。

你可以把它想象成一个快递分发中心：接收客户的请求，然后把请求分发到正确的地址。

## 什么是反向代理？

正向代理: 你 → 代理服务器 → 网站
- 你通过代理访问网站
- 网站看到的是代理的IP，不是你的IP
- VPN 就是正向代理

反向代理（Nginx 做的事情）： 用户 → 反向代理 → 真正的服务器
- 用户访问反向代理
- 反向代理帮你去找真正的服务器
- 用户不知道真正的服务器在哪里
