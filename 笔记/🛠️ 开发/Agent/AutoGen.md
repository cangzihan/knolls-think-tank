# AutoGen

https://github.com/microsoft/autogen

## Install
```shell
pip install ag2
```

## TaskWeaver与AutoGen的比较

| 特点     | TaskWeaver    | AutoGen       |
|--------|---------------|---------------|
| 适用任务类型 | 代码依赖任务，结构化管理  | 自动生成代码，多任务处理  |
| 主要功能 | 大型语言模型编码器  | 自动化工具，灵活性强  |
| 理想用途 | 数据和代码中心任务  | 多样化任务自动化  |
| 框架类型 | 单一代理框架  | 多代理系统  |
