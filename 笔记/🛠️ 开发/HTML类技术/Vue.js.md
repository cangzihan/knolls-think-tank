
# Vue.js

## 创建Vue.js项目
要在 Windows 电脑上运行和调试 Vue 网页，你需要搭建一个标准的 **前端开发环境**。别担心，这比装个大型游戏要简单得多。
以下是保姆级步骤：
### 第一步：安装“发动机” (Node.js)
Vue 运行在 Node.js 环境下。
 1. 前往 Node.js 官网。
 2. 下载 **LTS（长期支持版）**，点击安装包一路“下一步”即可。
 3. **验证**：按下 Win + R，输入 cmd，在黑色窗口输入：
   ```bash
   node -v
   npm -v
   
   ```
   看到版本号（比如 v20.x.x）就说明装好了。
### 第二步：安装“写代码的神器” (VS Code)
虽然记事本也能写，但 VS Code 是前端标配。
 1. 安装 Visual Studio Code。
 2. **关键插件**：在 VS Code 左侧插件栏搜索并安装 **Vue - Official** (以前叫 Volar)。它能让你的代码变彩色，并提示错误。
### 第三步：创建一个 Vue 项目
现在不用手动下载文件，用一行命令就能生成项目模板。
 1. 在电脑上找个文件夹（比如 D:\Projects），在地址栏输入 cmd 回车。
 2. 输入以下命令并回车：
   ```bash
   npm create vue@latest
   
   ```
 3. **跟着提示走**：给项目起个名字（比如 my-vue-app），剩下的选项（TypeScript, Router 等）如果是新手阶段，统统选 **No** 即可。
### 第四步：启动开发服务器
 1. 进入你的项目文件夹：
   ```bash
   cd my-vue-app
   
   ```
 2. 安装项目依赖（就像给电脑装驱动）：
   ```bash
   npm install
   
   ```
 3. **启动！**
   ```bash
   npm run dev
   
   ```
 4. 此时窗口会显示一个地址，通常是 http://localhost:5173/。按住 Ctrl 点击它，你的 Vue 网页就跑起来了！
### 第五步：如何调试？
#### 1. 实时预览（热更新）
在 VS Code 里打开你刚才创建的文件夹，修改 src/App.vue 里的文字并保存。你会发现浏览器**不用刷新**，内容直接就变了。
#### 2. 浏览器开发者工具 (F12)
在 Chrome 浏览器按 F12，点击 **Console (控制台)**。
 * 如果你在代码里写了 console.log(text.value)，结果会在这里打印出来。
 * 报错信息（红字）也会显示在这里，是你改 Bug 的主要依据。
#### 3. 强力推荐：Vue Devtools 扩展
去 Chrome 商店搜索并安装 **Vue.js devtools**。
安装后，F12 工具栏会多出一个“Vue”选项卡。
> **它能干啥？** 你可以直接在这里看到 ref 变量当前的值是多少，甚至手动修改它看网页的变化，而不需要去代码里改。
> 
### 总结流程
**装环境 (Node.js/VS Code) -> 命令行建项目 -> npm run dev -> 浏览器 F12 调试。**
如果你只是想随手写个小 Demo（就像你刚才在教程页面做的那样），不想装这么多东西，也可以直接用 StackBlitz 这种在线编辑器，它在浏览器里模拟了全套 Windows 环境。
