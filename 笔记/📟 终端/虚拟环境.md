## conda
```shell
conda env list
conda activate XXX
conda env remove --name your_env_name
conda create -n XXX python=3.8
```

使用 conda create 创建环境并指定路径
假设你想在 /path/to/your/env 创建一个新的Conda环境，可以使用以下命令：
```shell
conda create --prefix /path/to/your/env python=3.9
```

## uv
例程
```shell
# 1. 创建虚拟环境 (uv 会自动寻找系统中合适的 Python)
uv venv

# 2. 激活环境 (Ubuntu/Linux)
source .venv/bin/activate

# 3. 安装 Torch 2.9 (针对你刚装的 CUDA 12.8)
# uv 比 pip 聪明的地方在于它能极速解压，且支持通过 --index-url 避免下错 CPU 版本
uv pip install torch==2.9.0 torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
```
